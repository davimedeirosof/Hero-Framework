# Projetos — Índice

Registro de todos os projetos realizados com o Hero Framework.
Cada projeto tem seu próprio arquivo com contexto reutilizável.

## Template de projeto

Ao iniciar um novo projeto, crie `docs/projetos/nome-do-projeto.md` com:

```markdown
# Nome do Projeto

## Objetivo
Descrição do que foi feito.

## Agentes utilizados
- @agente-x — função exercida
- @agente-y — função exercida

## Skills utilizadas
- skill-nome — como foi usada

## Outputs gerados
- Arquivo/resultado 1
- Arquivo/resultado 2

## Contexto reutilizável
Padrões, decisões ou configurações que podem ser reaproveitados.

## Data
YYYY-MM-DD
```
