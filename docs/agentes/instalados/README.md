# Agentes Instalados — Índice

348 agentes instalados via Hero-Skills-Framework.
Localizados em `~/.claude/agents/`.

## Como usar

Chame diretamente pelo nome no Claude Code:
```
@nome-do-agente sua tarefa aqui
```

## Categorias principais

| Categoria         | Exemplos                                                    |
|-------------------|-------------------------------------------------------------|
| Desenvolvimento   | backend-developer, ai-engineer, angular-architect           |
| Segurança         | security-auditor, ad-security-reviewer                      |
| Qualidade         | architect-reviewer, architecture-auditor, code-review       |
| Documentação      | api-documenter, docs-writer                                 |
| DevOps            | build-engineer, azure-devops-specialist, azure-infra-engineer |
| Negócios          | business-analyst (em business/)                             |
