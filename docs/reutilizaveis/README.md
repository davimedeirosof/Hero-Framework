# Reutilizáveis — Índice

Prompts, padrões e contextos que podem ser reaproveitados em novos projetos.

## Como usar

1. Encontre o padrão desejado nesta pasta
2. Copie o conteúdo
3. Cole no início da sessão e diga ao Hero:
   _"use o padrão X abaixo como contexto"_

## Categorias

| Pasta | Conteúdo |
|-------|----------|
| `prompts/` | Prompts testados e validados por tipo de tarefa |
| `padroes/` | Padrões de arquitetura e estrutura de código |
| `contextos/` | Contextos de domínio para reusar entre projetos |
