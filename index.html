Pronto em 3 passos

Clone o repositório e execute o script. Tudo é configurado automaticamente.

bash
# Clone o repositório
$ git clone https://github.com/davimedeirosof/Hero-Skills-Framework.git

# Acesse o diretório
$ cd Hero-Skills-Framework

# Execute o instalador
$ ./install.sh

Instalacao completa!
01
Clone o repositório
Faça o clone do repositório Hero-Skills-Framework para sua máquina local via Git.
02
Execute o install.sh
O script faz backup do seu ~/.claude atual e instala todos os componentes: commands, agents, skills, hooks, persona Hero e Opensquad.
03
Abra o Claude Code
Abra uma nova sessão. O Claude responderá como Hero, com todos os 916 commands e 348 agentes prontos para uso.
04
Instale os plugins (opcional)
/plugin install planning-with-files@planning-with-files
/plugin install claude-hud
/plugin install oh-my-claudecode
